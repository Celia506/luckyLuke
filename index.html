Biographie

Salut tout le monde ! Moi, c'est Lucky Luke. Je suis né quelque part dans le Far West, là où les cactus poussent plus vite que les mauvaises herbes. Je suis un cow-boy solitaire, mais toujours prêt à faire régner la justice à travers les vastes plaines du Far West. Avec mon fidèle ami Jolly Jumper, on a affronté les pires bandits qui soient. Mon plus grand défi est de garder les frères Dalton derrière les barreaux, ce qui n'est pas une mince affaire ! Puis, sous le ciel étoilé, je gratte quelques accords sur mon banjo, après une journée bien remplie.

Oeuvres

Voici quelques-unes de mes réussites !

  • La Diligence (1968) : Un classique où Lucky Luke doit protéger une diligence remplie de passagers
  • Le Juge (1959) : Lucky Luke aide un juge honnête à faire régner la justice dans une ville corrompue.
  • Billy the Kid (1962) : Une aventure avec le célèbre hors-la-loi Billy the Kid.
  • Ma Dalton (1971) : Les frères Dalton tentent de se réhabiliter avec l'aide de Lucky Luke.
  • Calamity Jane (1967) : Lucky Luke rencontre la célèbre chasseuse de primes Calamity Jane.
  • L'escorte (1971) : Lucky Luke est chargé d'escorter un dangereux criminel.
